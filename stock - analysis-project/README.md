# 📊 Stock Analysis & Prediction Dashboard

## 🚀 Overview
This project is a complete stock analysis system that:
- Scrapes live stock data using Selenium
- Cleans and analyzes data using Pandas
- Visualizes trends using Matplotlib
- Provides an interactive dashboard using Streamlit
- Predicts stock prices using Machine Learning
- Performs basic sentiment analysis on news

---

## 🛠 Tech Stack
- Python
- Selenium
- Pandas
- Matplotlib
- Streamlit
- Scikit-learn
- yFinance

---

## 📊 Features
- 🔍 Search and filter stocks
- 📈 Top gainers & losers analysis
- 🤖 Buy/Sell recommendation
- ⚠️ Risk analysis
- 📊 Multi-stock comparison
- 📈 ML-based stock prediction
- 📰 News sentiment analysis
- ⬇️ Download dataset
