📊 Amazon Review Sentiment Analysis

🔍 Overview  :
               This project analyzes customer reviews to classify sentiment (Positive, Negative, Neutral) using NLP and machine learning techniques.

🎯 Objectives
Understand customer opinions from reviews
Build a sentiment classification model
Visualize insights using Power BI
🛠️ Tech Stack
Python (Pandas, NLP, Scikit-learn)
Machine Learning (Logistic Regression, TF-IDF)
Power BI (Dashboard)
Google Colab
📂 Dataset

Amazon product reviews dataset from Kaggle.

⚙️ Workflow
Data Cleaning & Preprocessing
Sentiment Labeling (based on ratings)
Text Processing (stopwords, lemmatization)
Feature Engineering (TF-IDF)
Model Training (Logistic Regression)
Evaluation (Accuracy, Precision, Recall)
Visualization (Power BI Dashboard)
📊 Key Insights
Majority of reviews are positive → high customer satisfaction
Negative sentiment concentrated in low ratings (1–2)
Strong alignment between ratings and sentiment
Neutral reviews are minimal → users have clear opinions
